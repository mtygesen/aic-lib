# Algorithms in C++
## General

To use the algorithms include the following header:

```cpp
#include "aic-lib/aic.hpp"
```

## Algorithms in this Library

### Searching

- Binary Search
- Linear Search
- Sentinel Search

### Sorting

- Bubble Sort
- Count Sort
- Heap Sort
- Insertion Sort
- Intro Sort
- Merge Sort
- Quick Sort
- Radix Sort
- Selection Sort
